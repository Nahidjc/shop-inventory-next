# Shop Inventory Management System (Frontend)

Welcome to the Shop Inventory Management System project! This is an open-source project template for building a shop inventory management system using Next.js.

## Project Description

This project aims to develop a comprehensive inventory management system for shops. It provides the following features:

- User Authentication: Users can sign up and log in to access the system.
- Inventory View: Users can view the current inventory of products.
- Add Products: Users can add new products with details like name, quantity, and price.
- Sale Products: Users can record sales by specifying the quantity and price of the sold products.
- Product Types: Support for various types of products.
- Sales Analytics:
  - Today's Sale Amount
  - Last 7 Days Sale Amount
  - Monthly Sale Amount
  - Bar Chart for Last 7 Days Sale Amount
  - Pie Chart for Product Sale Demand
  - Profit Calculation
  - Original Amount Calculation
  - Total Sale Amount Calculation
  - and more...

**New Features**:
- **Due Amount Tracking**: Keep track of due amounts for customers. Record and manage outstanding payments from customers for products purchased on credit.
- **Due Product Sales**: Track products sold on credit. Record sales where payment is due at a later date and manage product deliveries accordingly.
- **User Roles**: Implement user roles (e.g., admin, staff) with different access levels for enhanced security and management.
- **Product History**: Maintain a history of product changes, including additions, updates, and deletions, to track product lifecycle.
- **Supplier Management**: Manage suppliers, track supplier information, and streamline the process of restocking products.
- **Reports and Insights**: Generate reports for product sales, revenue, and inventory status to make informed business decisions.
- **Stock Alerts**: Set up alerts for low-stock products to avoid running out of popular items.
- **Localization**: Support multiple languages and currencies for a more diverse user base.
- **Barcode Scanning**: Enable barcode scanning for efficient product management.

## Getting Started

To set up and run the project locally, follow these steps:

1. Clone this repository to your local machine.
2. Install project dependencies: `npm install` or `yarn install`.
3. Configure your environment variables for authentication and database connections.
4. Start the development server: `npm run dev` or `yarn dev`.

## Project Structure

Here is an overview of the project structure:

- `pages/`: Next.js pages for different views and routes.
- `components/`: Reusable UI components.
- `api/`: Backend API calls and data handling.
- `public/`: Static assets like images and fonts.
